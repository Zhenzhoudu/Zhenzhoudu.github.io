# ZhenZhouDu-Homepage

这是一个基于企业级前端项目结构构建的个人学术主页工程。

## 目录结构

- `index.html`：页面结构
- `css/`：样式拆分为基础、布局、组件、动画、响应式、主题
- `js/`：主逻辑、语言、滚动、动画、夜间模式
- `lang/`：中文/英文文本资源
- `assets/`：图像、图标、字体资源

## 使用说明

1. 将项目部署到静态服务器（如 `Live Server` 或任意 Web 服务器）。
2. 打开 `index.html` 查看主页。

## 扩展方向

- 后续可新增 `lang/ja.json` 支持日语。
- 可将 `assets/images/` 替换为真实透明 PNG 素材。
- 可在 `index.html` 中补充论文、科研项目、博客、GitHub、Google Scholar、ORCID 等内容。
